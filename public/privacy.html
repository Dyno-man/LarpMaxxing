Plain-English privacy note

Keep the secrets
off the record.

Your screenshot is sent to the vision provider configured by the site operator. The default demo setup uses OpenRouter. LARPmaxxing does not save the uploaded image to disk or put it on the leaderboard.

If you choose to publish a score, the site stores your display name, optional X handle, score, rank, short roast, a few non-sensitive evidence labels, model mode, and timestamp.

Desktop screenshots can contain API keys, private messages, customer information, file names, and notifications. Remove or blur anything private before uploading. Do not submit an image you are not allowed to share.

This starter does not include accounts or a deletion dashboard. The server owner can remove leaderboard records directly from data/leaderboard.json.
